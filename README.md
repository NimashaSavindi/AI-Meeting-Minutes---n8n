# AI Meeting Minutes Generator (n8n + OpenAI)

## 🚀 Overview
This project automates meeting documentation using AI. It converts audio recordings into structured meeting minutes.

## ⚙️ Workflow
1. Upload MP3 audio
2. Transcribe using OpenAI Whisper
3. Generate structured meeting minutes
4. Convert to file
5. Upload to Google Drive

## 🛠️ Tech Stack
- n8n
- OpenAI API
- Google Drive API

## 📂 File
Import the workflow JSON into n8n to use.

## 📌 Use Case
Automates meeting note-taking for teams, businesses, and students.
